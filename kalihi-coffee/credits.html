Accessibility Statement

Our Commitment

Kalihi Coffee is committed to ensuring our website is accessible to all customers.

Accessibility Features

We have implemented the following features to improve accessibility:

  • Descriptive alt text on all images
  • Color contrast meeting WCAG AA standards (4.5:1 ratio for normal text, 3:1 for large text)
  • Keyboard navigation - all interactive elements can be accessed via keyboard
  • Semantic HTML structure with proper headings, landmarks, and form labels
  • Responsive design compatible with mobile devices and supports zoom up to 200%
  • Visible focus indicators for keyboard users

Testing

This website has been tested using:

  • WAVE Web Accessibility Evaluation Tool
  • Chrome Lighthouse accessibility audit
  • Manual keyboard navigation testing
  • Chrome DevTools responsive design mode

Our most recent Lighthouse accessibility score was 100/100.

Known Limitations

We are aware of the following accessibility limitations and are working to address them:

  • Our Contact form is lacking functionality

Feedback

If you encounter any accessibility barriers while using our website, please let us know. We will work to address issues promptly.
